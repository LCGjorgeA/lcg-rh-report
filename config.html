🗂 Categorías de Razón de Descarte
Estas categorías clasifican automáticamente la razón de descarte de cada candidato.
El dashboard lee el último comentario disponible y lo compara con las palabras clave de cada categoría. Las categorías se evalúan en orden — primera coincidencia gana. La comparación ignora mayúsculas y tildes.
💡 Cómo funciona: Para cada candidato descartado, el sistema toma el último comentario registrado (o la razón de descarte si no hay comentario) y lo busca en las palabras clave de cada fila, de arriba hacia abajo. Arrastra las filas para cambiar la prioridad.
